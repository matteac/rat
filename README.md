# rat
Bat without wings

## Installation

### Arch Linux
you can install rat from the AUR repository
```bash
yay -Sy rat
```


### From Source
First clone this repo
```bash
git clone https://github.com/matteac/rat.git
cd rat
```

Make
```bash
make
sudo make install
```

And now you can run it!

```bash
rat ~/path/to/some/file
```
